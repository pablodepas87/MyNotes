# C++ Switch

L'istruzione switch C++ esegue un'istruzione da più condizioni. È come l'istruzione ladder if-else-if in C ++.

    switch(expression){      
    case value1:      
    //code to be executed;      
    break;    
    case value2:      
    //code to be executed;      
    break;    
    ......      
        
    default:       
    //code to be executed if all cases are not matched;      
    break;    
    }
